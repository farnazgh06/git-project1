توی این برنامه یک فرم نظرسنجی طراحی شده که از کاربر اطلاعات شخصیش مثل نام و ایمیل و… و سطح اشنایی با نرم افزار رو میگیره و ازش سوالاتی راجب نرم افزار میپرسه

تغییرات کاربر اول :
اول زبان تک تک کامپوننت ها بخاطر دانشجوهای بین المللی را از فارسی به انگلیسی تغییر دادم
و سپس با استفاده از new Font("Arial",Font.BOLD, 13) فونتش رو به arial و به حالت bold و به سایز 13 تغییر دادم

تغییرات کاربر دوم :
کد اول و چهارم که با Setbounds() اجرا شده بودن رو با borderLayout جایگزین کردم 
و کد دوم و سوم را به BoxLayout تغییر دادم. 
بعد borderlayout کامپوننت هارو توی پنج ناحیه قرار میده و اندازه اشون به صورت خودکار تنظیم میشه.
سپس Boxlayout کامپوننت هارو توی یک خط عمودی یا افقی قرار میده.
سپس با استفاده از Color.Yellow رنگ پس زمینه هارو به زرد تغییر دادم 
و برای کد اول و دوم با استفاده از setHorizontal تراز افقی انجام دادم 
و توی وسط (Center) چیدمشون.‌ با استقاده از SetverticalAlignment تراز عمودی انجام دادم 
و در وسط چیدمشون.برای کد دوم و سوم که از Boxlayout توشون استفاده شده بود، 
با استفاده از new Dimension(width, height)، فاصله ثابت بین کامپوننت ها ایجاد کردم
